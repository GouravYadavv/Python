Pandas is a software library written for the Python programming language for data manipulation and analysis. In particular, it offers data structures and operations for manipulating numerical tables and time series. It is free software released under the three-clause BSD license.It is a Python package providing fast, flexible, and expressive data structures designed to make working with “relational” or “labeled” data both easy and intuitive.
It has 2 types of data structures 
1- Single-dimensioanl data structures (Series Object) 
2- Multi-dimensional data structures (Data Frame)
